--- Itens da pasta
Tanto word_counter quanto word_counter_2 são implementações do programa pedido.

O word_counter é um código mais fiel ao do livro.

O word_counter_2 é uma versão mais sofisticada, mais adaptada para a linguagem scala.

O text.txt é o arquivo que terá suas palavras contadas, e o stop_words é usado para remover as stop words no processamento de text.txt.

Fonte do text.txt: https://spectrum.ieee.org/tech-talk/computing/hardware/japans-fugaku-supercomputer-is-first-in-the-world-to-simultaneously-top-all-high-performance-benchmarks
Fonte do stop_words.txt: https://sites.google.com/site/kevinbouge/stopwords-lists

Kick Forward.pptx é o conjunto de slides utilizado para o vídeo.

--- Execução
Ambos programas foram feitos para serem executados no interpretador de scala.

Antes de executar o programa, é necessário alterar os endereços do text.txt e do stop_words.txt, que estão respectivamente nas linhas:
- 81 e 32 para o word_counter
- 106 e 53 para o word_counter_2
